# 🚀 LLMNightRun: LLM 실험 자동화 및 Agent 시스템

## 📌 프로젝트 개요

LLMNightRun은 LLM 실험을 자동화하고, CI/CD, 코드 리뷰, 프롬프트 반복 실험, Agent 워크플로우, MCP 기반 명령 시스템 등을 포함한 **통합형 LLM DevOps 플랫폼**입니다.  
LLM을 단순한 응답 도구가 아닌 **능동적인 실험 운영자(Agent)**로 변화시키는 것을 목표로 합니다.

---

## ✅ 구현된 전체 기능

### 1. LLM 자동 실행 및 실험 시스템

- 🌙 **심야 자동 프롬프트 실행 (NightRun)**  
  사용 요금이 낮은 심야 시간에 자동으로 프롬프트를 실행하고 응답을 저장합니다.

- 🗂️ **프롬프트 템플릿 관리 및 반복 실험**  
  다양한 실험 목적별 템플릿 구성 및 반복 실행 자동화.

- 🧾 **응답 자동 저장 및 메타 로그 기록**  
  프롬프트, 응답, 실행 시간, 모델명 등 로그 자동 기록.

---

### 2. 모델 실행 환경 자동화

- ⚙️ **환경 구성 자동화 (`setup.sh`, `config.yaml`)**  
  종속성 설치, 모델 다운로드, 구성 자동화.

- 🔄 **다양한 모델 플러그인 구조 지원**  
  GPT-4, Claude, Mistral, KoAlpaca 등 다양한 로컬/클라우드 모델 전환 가능.

- 🚀 **GPU 환경 자동 감지 및 전환**

---

### 3. 응답 평가 및 리라이팅 시스템

- 🧠 **로컬 평가 모델 연동을 통한 응답 품질 평가**
- ✍️ **자동 리라이팅 및 재질문 트리거**
- 🔁 **반복 실험 자동화 루프 구축**

---

### 4. LLM 기반 CI/CD 자동화

- 🧪 **LLM이 직접 코드 및 설정 분석/수정**
- 🛠️ `Dockerfile`, `GitHub Actions`, `yaml` 등 파일을 자동 개선 및 커밋
- 🚦 **배포 파이프라인 흐름 자동화**

---

### 5. MCP (Memory-Command-Process) 시스템

- 🧠 **stdin/stdout 기반 MCP 프로토콜 자체 구현**  
  자체 명령어 프로토콜을 통해 프롬프트 명령을 구조화된 실행 명령으로 변환합니다.

- 🌐 **웹 브라우저 콘솔 기반 MCP 서버 지원**  
  브라우저 상에서 명령어 입력 → 코드 실행 → 결과 출력의 과정을 실시간으로 관리.

- 🪟 **윈도우 콘솔 기반 MCP 서버 구현**  
  터미널에서 자연어 기반 명령 입력 가능 (예: "최근 실험 결과 정리해줘").

- 🧩 **프롬프트 → 명령 체계화 변환 기능**  
  사용자의 고수준 명령을 자동으로 실행 계획으로 분해하여 처리.

---

### 6. manus 기반 Agent 기능

- 🤖 **자연어 명령 기반 실험 파이프라인 자동 실행**
- 🔄 **워크플로우 단계 연결 및 결과 추적**
- 🎯 예: `"데이터 정제 → 모델 실행 → 결과 분석"` 등의 복합 명령 자동 실행

---

### 7. LLM 기반 GitHub 문서화 및 자동 업로드

- 📄 **LLM이 자동으로 README / 실험 요약 Markdown 문서 작성**
- ✍️ **커밋 메시지 및 파일 내용 요약도 자동 생성**
- ⬆️ **GitHub 자동 푸시 및 정리된 디렉토리 업로드**
