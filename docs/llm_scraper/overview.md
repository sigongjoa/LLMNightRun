# LLM Scraper: 웹 기반 LLM 출력 수집 시스템

## 개요

LLM Scraper는 OpenAI ChatGPT, Anthropic Claude, Google Gemini 등의 웹 UI 기반 대형 언어 모델(LLM)의 출력을 자동으로 크롤링하고 수집하는 도구입니다. API 호출이 제한적이거나 유료인 경우에도 동일한 프롬프트에 대해 여러 LLM의 출력을 비교 분석할 수 있습니다.

## 주요 기능

- **다중 LLM 웹 인터페이스 지원**: ChatGPT, Claude, Gemini 웹 인터페이스 자동화
- **세션 관리**: 브라우저 자동화를 통한 로그인 및 세션 관리
- **멀티 모델 비교**: 동일 프롬프트에 대한 여러 모델 응답 비교 수집
- **유연한 출력 형식**: JSON, CSV, Markdown, 텍스트 등 다양한 출력 형식 지원
- **배치 처리**: 여러 프롬프트 일괄 처리 지원

## 기술 스택

- **Python 3.8+**: 코어 언어
- **Selenium/Playwright**: 웹 브라우저 자동화
- **FastAPI**: (향후 확장) API 엔드포인트 제공
- **PyYAML**: 설정 관리
- **Pandas**: 데이터 처리 및 분석

## 아키텍처

LLM Scraper는 모듈식 아키텍처로 설계되었으며 다음 주요 구성 요소로 구성됩니다:

### 1. 코어 모듈

- **BaseScraper**: 모든 스크레이퍼의 기본 클래스 (브라우저 초기화, 탐색, 대기 등)
- **모델별 스크레이퍼**: ChatGPT, Claude, Gemini 등 모델별 특화 구현
- **설정 관리자**: 설정 로드 및 환경 변수 관리

### 2. 데이터 처리 모듈

- **DataProcessor**: 응답 데이터 처리 및 형식 변환
- **비교 엔진**: 다양한 모델의 응답 비교 및 분석

### 3. CLI 및 API 인터페이스

- **커맨드 라인 인터페이스**: 직관적인 명령어 기반 사용
- **API 엔드포인트** (향후 계획): 프론트엔드 및 외부 시스템 통합

## 작동 방식

1. **초기화**: 설정 및 자격 증명 로드
2. **브라우저 세션**: Selenium/Playwright를 통한 브라우저 제어
3. **인증**: 각 LLM 서비스에 자동 로그인
4. **프롬프트 제출**: 프롬프트 입력 및 제출
5. **응답 추출**: 응답 완료 대기 및 텍스트 추출
6. **처리 및 저장**: 결과 처리 및 지정된 형식으로 저장
7. **비교 분석**: (선택적) 여러 모델 응답의 비교 및 분석
