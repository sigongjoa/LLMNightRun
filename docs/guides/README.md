## 문서 작성 규칙

새로운 가이드 문서를 추가할 때는 다음 규칙을 따라주세요:

1. 파일명은 기능과 문서 유형을 나타내는 영문 소문자와 하이픈으로 작성 (예: `feature-guide.md`)
2. 첫 번째 헤딩(#)은 문서 제목으로 사용
3. 목차와 주요 섹션을 포함하여 구조적으로 작성
4. 코드 예제와 실행 방법을 포함
5. 문제 해결 가이드의 경우 원인과 해결 방법을 명확히 설명
