### GET /github-repos/

**List Repositories**

GitHub 저장소 목록 조회

Args:
    project_id: 프로젝트 ID (선택 사항)

#### 요청

**파라미터:**

| 이름 | 위치 | 타입 | 필수 | 설명 |
| ---- | ---- | ---- | ---- | ---- |
| project_id | query | any |  |  |

#### 응답

**상태 코드:** 200

설명: Successful Response

미디어 타입: `application/json`

**상태 코드:** 404

설명: Not found

**상태 코드:** 422

설명: Validation Error

미디어 타입: `application/json`

**상태 코드:** 400

설명: 잘못된 요청

미디어 타입: `application/json`

```json
{
  "error_code": "validation_error",
  "message": "데이터 검증 오류가 발생했습니다",
  "detail": [
    {
      "loc": [
        "body",
        "name"
      ],
      "msg": "필수 항목입니다"
    }
  ]

```

**상태 코드:** 401

설명: 인증되지 않음

미디어 타입: `application/json`

예시:
```json

  "error_code": "unauthorized",

  "detail": null

```

미디어 타입: `application/json`

예시:
