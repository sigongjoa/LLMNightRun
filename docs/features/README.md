# LLMNightRun 기능 문서

LLMNightRun의 주요 기능들에 대한 상세 문서입니다. 각 기능별로 API 엔드포인트, 구조, 사용 예시 등을 제공합니다.

## 통합 구성

LLMNightRun의 여러 기능들은 서로 통합되어 작동합니다:

1. **질문 및 응답 시스템**은 모든 기능의 기본이 되는 컴포넌트로, 사용자 입력을 처리합니다.
2. **코드 생성 시스템**은 질문 및 응답 시스템을 통해 전달된 요청을 분석하여 코드를 생성합니다.
3. **에이전트 시스템**은 질문 및 응답 시스템과 코드 생성 시스템을 활용하여 복잡한 작업을 수행합니다.
4. **인덱싱 시스템**은 문서와 코드를 분석하여 모든 시스템에서 사용할 수 있는 검색 기능을 제공합니다.
5. **자동 디버깅 시스템**은 코드 생성 시스템에서 생성된 코드나 사용자 코드의 문제를 해결합니다.
6. **내보내기 시스템**은 다른 모든 시스템에서 생성된 데이터를 외부로 내보냅니다.
7. **로컬 LLM 시스템**은 온라인 API 없이도 다른 모든 시스템이 작동할 수 있게 합니다.
8. **MCP 시스템**은 모든 LLM 모델의 동작을 중앙에서 제어합니다.

## API 문서

전체 API 문서는 `/api_doc.md` 파일 또는 실행 중인 서버의 `/docs` 엔드포인트를 통해 확인할 수 있습니다.
